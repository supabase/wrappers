# AWS DynamoDB Foreign Data Wrapper

This is a foreign data wrapper for [AWS DynamoDB](https://aws.amazon.com/dynamodb/). It is developed using [Wrappers](https://github.com/supabase/wrappers) and supports data scan and modify.
